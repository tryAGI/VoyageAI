# VoyageAI .NET SDK

Generated C# SDK for the [Voyage AI](https://www.voyageai.com/) API.

## Features

- Text embeddings (voyage-3, voyage-3-large, voyage-code-3, etc.)
- Multimodal embeddings (text + images via voyage-multimodal-3)
- Document reranking (rerank-2, rerank-2-lite)
- Microsoft.Extensions.AI `IEmbeddingGenerator` support
- NativeAOT/trimming compatible
